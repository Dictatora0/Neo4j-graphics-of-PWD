## 💡 常见问题

### Q: 如果中断了怎么办？

**A**: 直接重新运行 `./start.sh`，会自动从断点继续。

### Q: 如何停止运行？

**A**: 按 `Ctrl+C`，会自动保存进度。

### Q: 如何清除进度重新开始？

**A**:

```bash
rm -rf output/checkpoints/
./start.sh
```

### Q: 遇到错误怎么办？

**A**:

```bash
# 查看日志
tail -50 output/kg_builder.log

# 查看状态
./status.sh

# 查看修复总结
cat FIX_SUMMARY.md
```
