PWD 知识图谱系统 - 项目结构

根目录/
├── README.md                          # 主文档（整合所有关键信息）✅
├── config/                            # 配置文件
├── docs/                              # 技术文档目录
│   ├── MERGE_GUIDE.md
│   ├── MODEL_UPGRADE.md
│   └── UPGRADE_V2.5.md
├── scripts/                           # 工具脚本目录
│   ├── utils/                         # 工具函数
│   └── workflow/                      # 工作流脚本
├── output/                            # 输出目录
│   ├── concepts.csv                   # 抽取的概念
│   ├── relationships.csv              # 抽取的关系
│   └── kg_builder.log                 # 运行日志
├── archive/                           # 归档目录 📁
│   ├── README.md                      # 归档说明
│   ├── RUN.sh                         # 一键运行脚本 ⭐⭐⭐
│   ├── run_v2.5.sh                    # 交互式菜单 ⭐⭐
│   ├── start.sh                       # 简化启动 ⭐⭐
│   ├── test_imports.sh                # 导入测试 ⭐
│   ├── fix_all.sh                     # 自动修复 ⭐
│   ├── 其他脚本和文档...
│   
├── Python 源代码文件
│   ├── enhanced_pipeline.py           # 主流程（推荐）✅
│   ├── main.py                        # 完整流程（含Neo4j）
│   ├── concept_extractor.py           # LLM 概念抽取
│   ├── agentic_extractor.py           # Agentic Workflow
│   ├── graph_rag.py                   # GraphRAG 模块
│   ├── multimodal_extractor.py        # 多模态抽取
│   ├── concept_deduplicator.py        # BGE-M3 去重
│   └── 其他模块...

推荐使用方式:
1. 查看主文档: cat README.md
2. 直接运行: python enhanced_pipeline.py
3. 使用脚本: ./archive/RUN.sh
4. 查看归档: ls archive/

所有关键信息已整合到 README.md ✅
